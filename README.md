## TodoList Redux Toolkit

A todo application with add, delete, and filter features. The tech using React.js and to manage states, using Redux. Also styled using TailwindCss

## Features

- Add todo
- Delete todo
- Incomplete/Completed
- Filter todo

## Tech Stack

- React
- React Redux
- Tailwind Css
- React Icons
